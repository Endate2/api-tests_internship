# Задание 1: Баги на скриншоте

| # | Описание | Приоритет | Обоснование                                                                                                                                        |
|---|----------|-----------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 | **Ложное сообщение об ошибке** — при открытии списка объявлений появляется сообщение «Ошибка, обновите страницу» при корректной работе платформы | P1 — Высокий | Вводит пользователя в заблуждение: он может покинуть страницу. Напрямую влияет на пользовательский опыт.                                           |
| 2 | **Счётчик объявлений показывает 19, но фактически отображается 8** — рядом с заголовком указано «19», однако на странице видно 8 объявлений и кнопка «Показать 8 объявлений» | P1 — Высокий | Несоответствие между заявленным и фактическим количеством подрывает доверие к поиску.                                                              |
| 3 | **В категории «Кошки» присутствует объявление «Персидский котёнок плюшевый»** — игрушка, а не живое животное, попала в раздел с живыми питомцами | P2 — Средний | Нарушает корректность выдачи: нерелевантный товар попадает в тематическую категорию.                                                               |
| 4 | **Объявления из других городов отображаются при поиске «Котята в Москве»** — среди результатов есть объявление из Новосибирска | P1 — Высокий | Фильтрация по региону не работает. Пользователь ожидает локальные результаты — получает нерелевантные. Влияет на основную функцию поиска.          |
| 5 | **Неправильный тултип на кнопке лайка** — при наведении отображается «Не показывайте мне похожие объявления», хотя кнопка визуально является «Избранным»  | P2 — Средний | Вводит пользователя в заблуждение относительно действия кнопки. Некритично для функциональности, но создаёт путаницу в UX.                         |
| 6 | **Неверные варианты в фильтре «Возраст»** — в категории «Кошки» предлагается вариант «Щенок» вместо «Котёнок» | P2 — Средний | Ошибка в данных фильтра: термин из раздела «Собаки» попал в раздел «Кошки». Пользователь видит некорректный контент, но фильтр технически работает. |
| 7 | **Фильтр по полу не работает** — при выборе «Мальчик» в выдаче присутствуют объявления с полом «Девочка» | P1 — Высокий | Фильтрация по ключевому параметру не работает. Пользователь получает нерелевантные результаты.                                                     |
| 8 | **Фильтр по цене не работает** — задан диапазон 1 000–30 000 ₽, но в выдаче есть объявления за 40 000 ₽ и 500 ₽ | P1 — Высокий | Фильтрация по цене — одна из ключевых функций. Её неработоспособность приводит к нерелевантной выдаче.                         |
| 9 | **Кнопка «Назад» ведёт в неправильный раздел** — пользователь находится в «Кошки → Персидская», а кнопка возврата ведёт в «Животные» вместо «Кошки» | P2 — Средний | Нарушает навигацию: пользователь теряет контекст и вынужден проходить путь заново. Не блокирует, но ухудшает UX.                                   |
| 10 | **Неправильный порядок хлебных крошек** — отображается «Главная → Животные → Персидская → Кошки», хотя логичный путь: «Главная → Животные → Кошки → Персидская» | P2 — Средний | Хлебные крошки отражают неверную иерархию категорий. Дезориентирует пользователя при навигации.                                                    |
| 11 | **Неправильное название города в сортировке** — отображается «Сначала из Можайска» вместо «Сначала из Москвы» при выбранном регионе «Москва» | P1 — Высокий | Явное несоответствие выбранного региона, может привести к принятию неверных решений пользователем.                                                 |
| 12 | **Неравномерные отступы между объявлениями** — во второй строке между первым и вторым объявлением отступ заметно больше, чем между остальными | P3 — Низкий | Не влияет на функциональность, но ухудшает визуальное восприятие.                                                        |
| 13 | **Опечатка в хедере** — написано «Коръера в Авито» вместо «Карьера в Авито» | P2 — Средний | Орфографическая ошибка. Негативно влияет на репутацию бренда, хотя функциональность не затронута.                        |
| 14 | **Опечатка на кнопке** — написано «Уведомять о новых» вместо «Уведомлять о новых» | P2 — Средний | Орфографическая ошибка на интерактивном элементе. Ухудшает доверие к продукту.                                                                     |
| 15 | **Объявление «Экзотический короткошёрстный котёнок» не соответствует выбранной породе «Персидская»** — в выдаче по фильтру породы «Персидская» присутствует объявление другой породы | P1 — Высокий | Фильтр по породе не работает корректно: выдача содержит нерелевантные результаты. |
| 16 | **Кнопка «Показать 8 объявлений» не соответствует счётчику «19»** — внизу страницы кнопка показывает 8, а заголовок — 19 | P1 — Высокий | Противоречие между двумя элементами интерфейса. |
