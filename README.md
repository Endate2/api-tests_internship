
# API-тесты для проекта https://qa-internship.avito.com/api


## Запуск тестов

1.Подтяните необходимые зависимости через команду в корне проекта:

```bash
   go get -u -v -t all
```
2.Для запуска всех тестов достаточно выполнить команду в корне проекта:

```bash
  go test -v ./...
```

3.Отдельный тест можно запустить командой:

```bash
  go test <путь до go-файла с тестом>
```

> **_Windows:_** Все выполнять в  **powershell** оболоке
>

## Запуск линтера

1. Установить приложение [golangci-lint](https://golangci-lint.run/docs/welcome/install/local/)
2. Запуск линтера (из корня репозитория):

```bash
  golangci-lint run
```
