# BUGS.md

## BUG-001: Создание объявления с price = 0 возвращает 400

**Краткое описание:** Сервер отклоняет создание объявления с нулевой ценой, считая поле отсутствующим, хотя 0 — валидное значение (бесплатное объявление).

**Шаги воспроизведения:**
1. Отправить `POST /api/1/item` с телом:
```json
{
  "sellerID": 123456,
  "name": "Test",
  "price": 0,
  "statistics": { "likes": 1, "viewCount": 1, "contacts": 1 }
}
```

**Фактический результат:** `400 Bad Request`, тело: `{"result":{"message":"поле price обязательно","messages":{}},"status":"400"}`

**Ожидаемый результат:** `200 OK`, объявление создаётся с `price = 0`

**Серьёзность:** Major

**Окружение:** `https://qa-internship.avito.com`
